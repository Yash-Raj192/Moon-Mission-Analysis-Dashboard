<h2>🌑 Overview</h2>
<p>
  The Moon Mission Analysis Dashboard is an innovative data visualization solution, designed to explore and analyze comprehensive trends in lunar exploration from 1958 to 2023. Built from the ground up, this project delivers an interactive dashboard environment to uncover key insights surrounding the global race to the moon.
</p>

<h2>✨ Key Features</h2>
<ul>
  <li><strong>Total Mission Analytics:</strong> Monitor the complete count and distribution of moon missions across decades.</li>
  <li><strong>Mission Outcome Breakdown:</strong> Visualize performance metrics such as success rates, operational status, and failure classifications.</li>
  <li><strong>Mission Type Insights:</strong> Explore orbiter, lander, rover, flyby, impactor, and sample return missions in detail.</li>
  <li><strong>Country Contributions:</strong> Track and compare lunar exploration efforts across countries and agencies.</li>
  <li><strong>Rocket Carrier Analysis:</strong> Identify the most utilized launch vehicles for moon missions worldwide.</li>
  <li><strong>Annual Trends:</strong> Analyze launches by year to observe technological progress and shifting space strategies.</li>
  <li><strong>Rich Visual Design:</strong> Engaging charts, interactive slicers, and high-impact visuals for deeper data understanding.</li>
</ul>

<h2>🛠️ Tools & Technologies</h2>
<ul>
  <li>Power BI for advanced data visualization and dashboard creation</li>
  <li>Data cleaning, validation, and transformation</li>
  <li>Custom Excel/CSV data preparation and integration</li>
  <li>Interactive reports for dynamic filtering and analysis</li>
</ul>

<h2>🚀 How It Works</h2>
<ol>
  <li>Import the custom moon mission dataset (CSV format) into Power BI.</li>
  <li>Explore the interactive dashboard pages covering outcomes, mission types, years, countries, and rocket carriers.</li>
  <li>Leverage slicers and filters to tailor exploration by year or mission outcome.</li>
  <li>Utilize the visual storytelling approach to communicate trends to the audience.</li>
</ol>

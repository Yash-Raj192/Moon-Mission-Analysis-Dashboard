<h2>Project Overview</h2>
<p>
  The Moon Mission Analysis Dashboard is a comprehensive, interactive dashboard built in Power BI that visualizes the history of lunar exploration by multiple countries and space agencies.<br>
  It provides deep insights into the outcomes, timelines, mission types, and the technological evolution of moon missions using a rich dataset spanning more than six decades.
</p>

<h2>Features</h2>
<ul>
  <li>Total count and timeline of all moon missions since 1958</li>
  <li>Analysis by mission outcome (success, failure, operational, partial, en route)</li>
  <li>Breakdown by mission type (lander, orbiter, flyby, impactor, rover, etc.)</li>
  <li>Contribution by country and rocket carrier</li>
  <li>Trends by year and top rocket carriers used</li>
  <li>Interactive slicers for year, mission outcome, type, and country</li>
  <li>Professional, modern visuals for maximum data clarity</li>
</ul>

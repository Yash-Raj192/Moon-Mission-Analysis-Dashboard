<h2>🌑 Project Overview</h2>
<p>
  The Moon Mission Analysis Dashboard is a comprehensive Power BI analytics project developed to study historical lunar exploration missions conducted between 1958 and 2023. The dashboard provides a multi-page interactive analytical environment for exploring mission trends, success rates, rocket carrier usage, country-wise participation, and decade-wise evolution of moon missions.
</p>

<p>
  This project focuses not only on visualization, but also on data preprocessing, feature engineering, KPI generation, and analytical storytelling using Power BI and DAX measures.
</p>

<h2>✨ Key Features</h2>
<ul>
  <li><strong>Executive Dashboard Overview:</strong> KPI cards displaying total missions, successful missions, countries involved, and overall mission success rate.</li>

  <li><strong>Mission Outcome Analysis:</strong> Visual representation of successful, failed, and partially successful moon missions.</li>

  <li><strong>Country-Wise Mission Analysis:</strong> Comparative analysis of top contributing countries in lunar exploration.</li>

  <li><strong>Mission Trend Analysis:</strong> Year-wise and decade-wise visualization of moon mission growth over time.</li>

  <li><strong>Rocket Carrier Insights:</strong> Identification and comparison of the most frequently used rocket carriers.</li>

  <li><strong>Mission Era Segmentation:</strong> Categorization of missions into historical space exploration eras for analytical storytelling.</li>

  <li><strong>Interactive Filtering:</strong> Dynamic slicers enabling filtered analysis based on mission years.</li>

  <li><strong>Custom Dashboard Theme:</strong> Dark-themed dashboard with visually engaging layouts, charts, and analytical sections.</li>
</ul>

<h2>🛠️ Tools & Technologies Used</h2>
<ul>
  <li><strong>Power BI</strong> — Dashboard development and visualization</li>

  <li><strong>Power Query</strong> — Data cleaning and preprocessing</li>

  <li><strong>DAX (Data Analysis Expressions)</strong> — KPI measures and calculated metrics</li>

  <li><strong>CSV Dataset Integration</strong> — Historical moon mission dataset processing</li>

  <li><strong>Interactive Visual Analytics</strong> — Trend analysis, filtering, and comparative insights</li>
</ul>

<h2>🧹 Data Preprocessing & Feature Engineering</h2>
<p>
  Multiple preprocessing and transformation steps were performed before dashboard creation to improve analytical depth and reporting quality.
</p>

<ul>
  <li>Creation of <strong>Decade</strong> column for decade-wise mission analysis</li>

  <li>Creation of <strong>Mission Era</strong> categories for historical segmentation</li>

  <li>Mission outcome categorization into:
    <ul>
      <li>Success</li>
      <li>Failure</li>
      <li>Partial Success</li>
    </ul>
  </li>

  <li>Country extraction and normalization for country-wise analytics</li>

  <li>Custom DAX measures for:
    <ul>
